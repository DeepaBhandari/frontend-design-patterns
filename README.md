# 🚀 Frontend Systems Playground

A compact Next.js application showcasing **production-grade frontend patterns** commonly used in scalable SaaS products.

> Built based on real-world experience working on multi-tenant platforms, admin panels, and complex frontend systems.

---

## 🧠 What this project demonstrates

### 🔌 API State Management

* Data fetching with React Query
* Handling loading, error, and success states
* Retry mechanisms

### 🧾 Form Validation

* Schema-based validation using Zod
* React Hook Form integration
* Real-time validation UX

### 🎛️ Feature Flags / Role-Based UI

* Conditional rendering based on user roles
* Simulated RBAC (Admin vs User)

### 🔍 Debounced Search

* Optimized API calls with debouncing
* Improved performance and user experience

---

## 🛠️ Tech Stack

* Next.js (App Router)
* React + TypeScript
* Tailwind CSS
* React Query (TanStack Query)
* React Hook Form + Zod
* Axios
* pnpm
